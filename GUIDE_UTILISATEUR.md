# Guide Utilisateur - Analyseur d'Espace Disque

## Lancement rapide

Un raccourci a ete automatiquement cree sur votre bureau :

```
Bureau\OUTILS_DI\Analyseur_Espace_Disque.vbs
```

**Double-cliquez sur ce fichier** pour lancer l'analyseur.

---

## Fonctionnement

1. **Double-cliquez** sur `Analyseur_Espace_Disque.vbs`
2. L'interface graphique s'ouvre (sans fenetre noire)
3. L'analyse de votre profil demarre automatiquement

---

## Onglets disponibles

| Onglet | Description |
|--------|-------------|
| Vue d'ensemble | Statistiques globales de votre espace disque |
| Hierarchie | Arborescence des dossiers avec leurs tailles |
| Top 20 Fichiers | Les 20 plus gros fichiers de votre profil |
| Doublons | Detection des fichiers en double |
| Nettoyage | Assistant pour liberer de l'espace |

---

## Nettoyage guide

1. Allez dans l'onglet **Nettoyage**
2. Selectionnez les elements a nettoyer (cache navigateur, fichiers temporaires...)
3. Cliquez sur **Analyser** pour voir l'espace recuperable
4. Cliquez sur **Nettoyer** pour supprimer les fichiers

---

## En cas de probleme

Si le lanceur ne fonctionne pas :

1. Verifiez que le dossier `OUTILS_DI` existe sur votre bureau
2. Verifiez que `DiskSpaceAnalyzer.ps1` est present dans ce dossier
3. Contactez le support informatique

---

## Raccourcis clavier

- **F5** : Relancer l'analyse
- **Ctrl+E** : Exporter le rapport HTML
